Beginner JavaScript web setup

Install VS Code, Live Server, and Git

This setup gives you a local editor, a browser preview workflow, and Git source control for course examples and repository work. You do not need Node.js for this course.

!
Install only what this course needs. This guide covers VS Code, Live Server, Git, and a modern browser. Save Node.js, npm, Vite, and frameworks for the next course.

Setup Checklist

  1. Install Visual Studio Code

    Go to code.visualstudio.com/download, download VS Code for your operating system, and run the installer.

  2. Install Live Server

    Open VS Code, select Extensions, search for Live Server, and install the extension by Ritwick Dey. Direct link: Live Server.

  3. Install Git

    Go to git-scm.com/downloads and install Git. Restart VS Code after installation so it can detect Git.

  4. Configure Git identity

    Open Terminal, Command Prompt, or PowerShell and set the name and email address Git will attach to your commits.

  5. Confirm Git in VS Code

    Open a course folder in VS Code, select Source Control, and confirm VS Code sees your file changes. VS Code has Git support, but Git still must be installed.

Git Commands

  • Verify Git, configure your identity, and check your settings:
git --version git config --global user.name "Your Name" git config --global user.email "your-email@example.com" git config --global user.name git config --global user.email

VS Code Git Test

  • Open a folder in VS Code and create a small file.
  • Select Source Control. You should see the changed file.
  • Type a short message and click Commit.
practice.html Commit message: setup test

Live Server Test

  • Create a folder and add index.html.
  • Right-click the file and choose Open with Live Server, or click Go Live.
<h1>Hello, web!</h1> <script> console.log("Live Server works"); </script>
Browser Use a modern browser such as Chrome or Edge. We will use browser DevTools throughout the course.
VS Code Git If VS Code says Git is missing, restart VS Code after installing Git. If needed, restart the computer.
No Node Do not install Node.js for this course unless your instructor tells you to. That comes later.

When you finish: you are ready to edit web files locally, preview them with Live Server, and use Git with VS Code for course repository work.
